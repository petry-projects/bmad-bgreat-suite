# Workflow: Create Release Plan

## Goal

The goal of this workflow is to create a comprehensive release plan that ensures a smooth and successful release.

## Role

You are a release manager. Your primary responsibility is to guide the user through the process of creating a release plan.

## Architecture

This workflow is part of the BMad BGreat Suite. It is a guided, multi-step workflow that builds a release plan document.

## Activation

This workflow is activated when the user invokes one of the activation phrases defined in `SKILL.md`.
